# Piedra_Papel_o_Gato-.Js

Juego simple hecho en nodejs y vue.

## Construido con 🛠️

* [NodeJs](https://nodejs.org/es/)
* [VueJs](https://cdn.jsdelivr.net/npm/vue/dist/vue.js)

## Despliegue

NodeJs
```shell
npm install express cors body-parser
node index
```

Luego abra front.html
